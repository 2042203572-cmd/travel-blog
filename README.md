
# 探索世界 - 旅游博客

一个响应式的旅游博客网站，展示各地的旅游景点和游记。

## 功能特性

- 🌙 深色/浅色主题切换
- 🎠 自动轮播图片展示
- 📱 完全响应式设计
- 📝 博客文章展示与详情
- 💬 评论系统（本地存储）
- 📱 移动端友好导航
- ⚡ 平滑页面切换

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **架构**: 模块化JavaScript，单页面应用(SPA)风格
- **存储**: 浏览器本地存储(localStorage)
- **样式**: CSS变量，Flexbox，Grid布局
- **工具**: 原生JavaScript，无框架依赖
